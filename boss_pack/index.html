How To Use This Pack

  1. Start in the Excel pack, not the JSON files.
  2. Read the Priority Research tab for the first worklist.
  3. Check Evidence Gate before treating any score seriously.
  4. Fill the Review Decisions tab with advance, watch, decline, contact, meeting, or diligence decisions.
  5. Use Source Coverage and Data Room tabs to see what public evidence still needs review.
